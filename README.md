# NoteManagementApp

Not Yönetim Sistemi

Kullanıcı Notlar ve iç notlar ekleme - Silme - Güncelleme - Listleme ve not detay yapabilir.
API ve MVC olamak üzere iki proje var, MVC API İLE MsSQL'den iletişime geçiyor.
